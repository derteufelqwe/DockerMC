# Container labels
All containers which belong to DockerMC have the tag `Owner=DockerMC`

## Basic container types
Basic container types have the key `Type` and the following values:
- REGISTRY
- DNS
- SYSTEM_CERTS_GEN
- BUNGEE
- MINECRAFT
